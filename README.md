# Retro Shooter

Small fullscreen 2D shooter built with Pygame. Features:

- Fullscreen scaling
- Player, enemies, obstacles, bullets
- Lives displayed as hearts
- Score and high score persistence
- Power-ups: double shot and invisibility
- Game Over screen with restart

How to run

1. Install dependencies:

```powershell
python -m pip install pygame
```

2. Run:

```powershell
python main.py
```

Notes

- Add optional assets `power_double.png`, `power_invis.png`, and `pickup.wav` to enable icons and sounds for powerups.
- The project doesn't include an automated build system.
